I have several projects written in Processing! All of these were done as "quick tests" just to get the ball rolling and help me grasp the underlying concepts involved.<br>

### [Octrees](https://github.com/MaxTheMooshroom/Octree)
An implementation of Quad-Trees for 3d space

### [Spherical Geometry](https://github.com/MaxTheMooshroom/spherical_geometry)
An implementation of the full pythagorean theorem with varying constants. You can change the variables live and observe the changes. 

### [Star Field](https://github.com/MaxTheMooshroom/star_field)
When a spaceship goes into "hyper-space" (or an equivalent sci-fi trope), the stars typically move in a specific way. This is an attempt to recreate that. 

### [Perlin Flight](https://github.com/MaxTheMooshroom/perlin_flight)
Fly over procedurally generated terrain using perlin noise. 

### [Menger Sponge](https://github.com/MaxTheMooshroom/menger_sponge)
A menger sponge. 

### [Flocking](https://github.com/MaxTheMooshroom/flocking)
An implementation of the boyd algorithm.
