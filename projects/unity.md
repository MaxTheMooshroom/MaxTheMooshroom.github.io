# Unity

A Unity project I'm working on is ecosystem simulation!
As of now (October 21st, 2019) there is no public source code.<br>
When [Splice](https://maxthemooshroom.github.io/projects/cpp/) is far enough in development to do so, these projects will likely be moved to work on that engine instead. 


### Ecosystem Simulation
I plan on creating a food web with various plants and animals that have traits similar to real-life counterparts in an attempt to create a statistical analysis of what it takes to create a sustainable habitat for long term survival, and what impacts human activities have on that stability. <br>
